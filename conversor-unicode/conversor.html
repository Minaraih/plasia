Conversor Unicode

Para o que serve?

O conversor unicode serve para transferir a escrita latina para números unicode ou de unicode para letras latinas.

Como usar?

Digite as letras no campo de digitação depois clique em enviar, e não se esqueça de selecionar a opção unicode, para converter de letras para unicode. Também é possível transferir de Unicode para letras, apenas altere a seleção para Texto.
